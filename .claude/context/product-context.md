# Product Context

## User Personas

### Primary: Personal OKR Practitioner
- Sets quarterly/yearly/N-yearly life themes with objectives and key results
- Assigns daily focus to themes via the calendar
- Breaks objectives into actionable tasks on the EisenKan board
- Reviews progress by navigating between views

## Core Use Cases

### 1. Strategic Planning (OKR View)
- Create and organize life themes (Health, Career, Learning, etc.)
- Define objectives under themes with nested structure
- Set measurable key results with progress tracking

### 2. Daily Focus Assignment (Calendar View)
- Assign a theme focus for each day of the year
- Visualize focus distribution across themes via color coding
- Navigate to related tasks for a specific day/theme

### 3. Task Execution (EisenKan View)
- Create tasks linked to themes with Eisenhower priority
- Move tasks through kanban columns
- Automatic priority promotion based on dates
- Filter tasks by theme, tags, or date

### 4. Cross-View Navigation
- Click a theme in EisenKan to navigate to OKR view
- Click a date in EisenKan to navigate to Calendar view
- Click a task indicator in Calendar to navigate to EisenKan
- Breadcrumb trail for navigation history

## Pain Points Addressed
- Disconnected planning tools (separate apps for goals, calendar, tasks)
- Lack of visibility into how daily work connects to long-term goals
- No local-first alternative to cloud-based planning tools
