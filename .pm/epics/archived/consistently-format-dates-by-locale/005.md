# Task: Migrate CalendarView date displays and locale-aware names

## Description
Replace hardcoded English month names, weekday names, and date formatting in `CalendarView.svelte` with calls to the shared `date-format.ts` utility, making the calendar fully locale-aware.

## Acceptance Criteria
- [ ] `monthNames` array replaced by `formatMonthName()` calls
- [ ] `shortMonthNames` array replaced by `formatShortMonthName()` calls
- [ ] `weekdayNames` array replaced by `formatWeekdayShort()` calls
- [ ] `displayDate()` function uses `formatDate()` from the utility
- [ ] No hardcoded English month/weekday name arrays remain in CalendarView
- [ ] Existing CalendarView tests pass (update assertions if needed)
- [ ] Calendar renders correctly in both native and browser-based app

## Technical Details
- Import formatting functions from `../lib/utils/date-format`
- Replace `monthNames` constant: derive from `formatMonthName(i)` for i in 0..11
- Replace `shortMonthNames` constant: derive from `formatShortMonthName(i)` for i in 0..11
- Replace `weekdayNames` constant: derive from `formatWeekdayShort(i)` for i in 0..6
- Replace `displayDate(month, day)` body: construct ISO string, pass to `formatDate()`
- Note: `formatDate(month, day)` (which returns ISO strings for internal use) should remain unchanged â€” it's for data, not display
- Files affected: `frontend/src/views/CalendarView.svelte`, `frontend/src/views/CalendarView.test.ts`
