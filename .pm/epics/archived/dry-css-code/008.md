# Task: Replace all remaining hardcoded hex colors with design tokens

## Description
Sweep through all Svelte component `<style>` blocks and replace every remaining hardcoded hex color value with the corresponding CSS custom property from the design token palette defined in Task 001. This is a mechanical find-and-replace task.

## Acceptance Criteria
- [ ] Zero hardcoded hex color values in `<style>` blocks of Svelte components (excluding Wails-generated files in `frontend/src/lib/wails/`)
- [ ] All replacements use the correct token (e.g., `#2563eb` → `var(--color-primary-600)`)
- [ ] Hex values inside `color-mix()`, `rgba()`, or inline `style=` attributes are handled appropriately (use tokens where possible, document exceptions)
- [ ] Existing `--theme-color` system preserved and compatible
- [ ] Visual appearance unchanged
- [ ] Both Wails native and browser mock mode work correctly
- [ ] `make frontend-lint` and `make frontend-check` pass
