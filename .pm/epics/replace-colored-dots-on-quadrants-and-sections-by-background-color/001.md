# Task: Replace colored dots with header backgrounds and container tints

## Description

In both `EisenhowerQuadrant.svelte` and `EisenKanView.svelte`, replace the small colored dot elements with colored header backgrounds and a subtle color tint on the container body.

## Acceptance Criteria

- [ ] `.color-indicator` span removed from `EisenhowerQuadrant.svelte` markup and CSS
- [ ] `.section-color` span removed from `EisenKanView.svelte` markup and CSS
- [ ] Quadrant headers have background color matching the quadrant's priority color
- [ ] Section headers have background color matching the section's color
- [ ] Header text (title, count badge) is readable against colored backgrounds (white text)
- [ ] Quadrant container body has a subtle tint of the priority color
- [ ] Section container body has a subtle tint of the section color
- [ ] Task cards remain white and clearly readable against tinted backgrounds
- [ ] Drag-and-drop still works in both components

## Technical Details

**`EisenhowerQuadrant.svelte`:**
- Remove `.color-indicator` span
- Set `--quadrant-color` CSS custom property on `.quadrant` via inline style
- Update `.quadrant-header` to use `background-color: var(--quadrant-color)` with white text
- Update `.quadrant` background to use a subtle tint via `color-mix()` or low-opacity
- Update `.task-count` and `.quadrant-title` for white text
- Delete `.color-indicator` CSS

**`EisenKanView.svelte`:**
- Remove `.section-color` span
- Set `--section-color` CSS custom property on `.column-section` via inline style
- Update `.section-header` to use `background-color: var(--section-color)` with white text, add padding and border-radius
- Update `.column-section` background to use a subtle tint
- Update `.section-count` and `.section-title` for white text
- Delete `.section-color` CSS
