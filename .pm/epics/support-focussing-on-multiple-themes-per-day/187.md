# Task: Frontend multi-theme support + gradient visualization

### Gradient helper
Create a helper function `themeColorsToBackground(colors: string[]): string`:
- 0 colors: return `''` (use default/Sunday background)
- 1 color: return `background-color: {color}20;`
- N colors: return `background: linear-gradient(to right, {c1}20 0%, {c1}20 {w-2px}, {c2}20 {w}, {c2}20 {2w-2px}, ...);`
- Band width = `100% / N`, with ~2px gradient transitions

### Dialog change
The key change is in the `onThemeSelectionChange` callback:
```
// Before (forces single theme):
editThemeId = ids.length > 0 ? ids[ids.length - 1] : '';
// After (accepts all themes):
editThemeIds = ids;
```

### App.svelte change
- `todayFocusThemeId` → `todayFocusThemeIds`
- `resolveTodayFocus` reads `focus.themeIds` (array)
- Today-focus toggle: `filterThemeIds = todayFocusThemeIds`
